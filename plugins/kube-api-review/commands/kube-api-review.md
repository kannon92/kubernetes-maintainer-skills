## Name

kube-api-review:kube-api-review

## Synopsis

Review Kubernetes API type definitions against community conventions.

## Description

Reads the staging directory of a PR and reviews the types files against Kubernetes API conventions.

## Implementation

Please read the staging directory of the PR and read the types files.

The review conventions should follow https://github.com/kubernetes/community/blob/master/contributors/devel/sig-architecture/api-conventions.md
